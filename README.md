# HCCheck — 运管站货车审验自动化工具

> 自动完成运管站货车年审流程，支持带挂/不带挂两种流程类型。

![Python](https://img.shields.io/badge/Python-3.12+-blue)
![License](https://img.shields.io/badge/license-MIT-green)

---

## 🎯 功能

- **批量审验** — 自动从年审列表取车牌，一辆接一辆处理
- **两种流程** — 支持带挂车辆和不带挂车辆
- **工作台续跑** — 刷新页面后自动检测未完成任务，从中断处继续
- **自动登录** — 填入账号密码后自动登录，无需手动操作
- **GUI 界面** — Tkinter 图形界面，配置简单，一键启动
- **定时任务** — 支持配置定时自动启动（Cron 表达式）
- **打包部署** — 单 exe 文件，拷到运管站电脑直接运行

---

## 📋 系统要求

- Windows 7/10/11
- Google Chrome 浏览器（已安装）
- Python 3.12+（开发用，运管站电脑跑 exe 不需要）

## 🚀 快速开始

### 方式一：运行 exe（推荐）

1. 下载 `dist/HCCheck.exe`
2. 双击运行
3. 设置页填入账号密码 → 点"保存所有设置"
4. 运行页点"启动"

### 方式二：运行源码

```bash
# 安装依赖
pip install -r requirements.txt
python -m playwright install

# 运行
python gui.py
```

### 打包 exe

```bash
pip install pyinstaller
python build.py
# 输出: dist/HCCheck.exe
```

## ⚙ 配置说明

| 配置项 | 说明 |
|--------|------|
| 登录账号 | 运管站系统账号 |
| 登录密码 | 运管站系统密码 |
| 自动登录 | 填完账号密码后自动点登录按钮 |
| 最多跑几辆 | 0 = 无限，直到列表为空 |
| 定时任务 | Cron 表达式，如 `0 8 * * *` = 每天8点 |
| 无头模式 | 后台运行，不显示浏览器窗口 |
| 操作间隔 | 每步操作之间的等待时间（秒） |

## 🔄 审验流程

### 不带挂（普货）

```
车辆检测 → 技术岗位审核 → 业务岗位审核 → 车辆年审 → 归档
```

### 带挂（挂车）

```
车辆检测 → 业务岗位审核 → 车辆年审1 → 车辆年审2 → 归档
```

## 🐛 常见问题

**Q: 运管站电脑没有 Python 怎么办？**
A: 直接运行 `HCCheck.exe`，不需要 Python 环境。

**Q: 界面显示中文乱码？**
A: 确保 Windows 系统语言设为中文，或使用支持中文的字体。

**Q: 某辆车一直失败怎么办？**
A: 失败 3 次后自动加入黑名单，不再重试。需手动在日志中排查原因。
