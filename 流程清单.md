# HCCheck — 流程清单

## 整体流程

```
启动 HCCheck.exe
  ↓
① 打开浏览器 → 登录页面
  ↓
② 登录（账号密码自动填入 / 手动登录）
  ↓
③ 导航树自动检测 → 切换到工作台
  ↓
╔══════════════════════════════════════════╗
║           主循环 (while True)            ║
╠══════════════════════════════════════════╣
║ ① 检查工作台是否有未完成任务（续跑）      ║
║    ├ 有 → 读当前节点 → 从对应步骤开始    ║
║    └ 无 → 去普货审验列表取车牌 → 创建    ║
║ ② 车牌含"挂"? → 带挂流程 / 不带挂流程    ║
║ ③ 按节点循环处理各弹窗                    ║
║ ④ 完成 → 回工作台 → 下一辆               ║
╚══════════════════════════════════════════╝
  ↓
列表为空 → 流程结束
```

---

## 不带挂流程 (process_unmarked)

| 步骤 | 节点 | 操作 |
|------|------|------|
| popup1 | 车辆检测 | 提交 → 弹窗(全选→确定) |
| popup2 | 技术岗位审核 | 打印告知单(4种策略) → 关预览 → 提交 → 弹窗 |
| popup3 | 业务岗位审核 | 提交 → 弹窗 |
| popup4 | 车辆年审 | 点"年度审验" → 提交 → 弹窗 |
| popup5 | 归档 | 完成 → 2个confirm |

## 带挂流程 (process_marked)

| 步骤 | 节点 | 操作 |
|------|------|------|
| popup1 | 车辆检测 | 提交 → 弹窗(全选→确定) |
| popup2 | 业务岗位审核 | 提交 → 弹窗（跳过打印） |
| popup3 | 车辆年审1 | 弹窗 |
| popup4 | 车辆年审2 | 点"年度审验" → 提交 → 弹窗 |
| popup5 | 归档 | 完成 → 2个confirm |

## 续跑逻辑

工作台读到当前节点后直接映射到步骤常量，跳过前面所有已完成步骤：

| 工作台节点 | 从哪步开始 |
|-----------|-----------|
| 车辆检测 | popup1 |
| 技术岗位审核 | popup2 |
| 业务岗位审核 | popup3 |
| 车辆年审 | popup4 |
| 归档 | popup5 |
