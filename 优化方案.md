# HCCheck 代码优化方案

> **原则**：每步独立可测试，出问题能回退
> **策略**：保留旧函数 → 写新函数 → 对比测试 → 切换 → 删旧函数

### Step 2.4：抽 `extract_plates_from_table(table)`

**应用场景：**
- `get_next_plate_from_list` 里扫描 table
- `main()` 里扫描工作台 table

**新代码：**
```python
def extract_plates_from_table(table, skip_plates=None):
    """从 table 中提取所有车牌号(可选过滤黑名单)
    Returns: [(plate, row_text, cells), ...]
    """
    if skip_plates is None:
        skip_plates = SKIP_PLATES
    results = []
    try:
        rows = table.locator("tbody tr")
        for r in range(rows.count()):
            cells = rows.nth(r).locator("td")
            row_texts = [cells.nth(j).text_content().strip() for j in range(cells.count())]
            row_full = " ".join(row_texts)
            for t in row_texts:
                if config.PLATE_RE.match(t) and t not in skip_plates:
                    results.append((t, row_full, cells))
                    break
    except:
        pass
    return results
```

**测试方式：**
- 跑批前先调用一次，打印提取到的车牌数
- 与原代码结果对比

**回退：** 改回原来的内联代码

## 🎯 阶段三：拆 main() 函数（中等风险，逐步进行）

### Step 3.1：抽 `_check_and_process_workbench()`

**当前 main() 第一段（工作台检查 + 续跑）：**
```python
# 在 main() 顶部嵌套 ~50 行
workbench_plates = []
for r in range(...):  # 扫描工作台
    ...
if workbench_plates:
    for plate, current_node, flow_name in workbench_plates:
        ...  # 处理每个任务
    continue
```

**重构为：**
```python
def _check_and_process_workbench(page, processed, failures):
    """检查工作台并处理续跑任务
    Returns: (processed, should_continue_loop)
    """
    workbench_plates = _scan_workbench(page)
    if not workbench_plates:
        return processed, True  # 工作台空，回到主循环取新车
    
    print(f"\n📋 工作台有 {len(workbench_plates)} 个待办任务:")
    for p, n, f in workbench_plates:
        print(f"  · {p} ({f} | {n})")
    
    for plate, current_node, flow_name in workbench_plates:
        if plate in SKIP_PLATES:
            continue
        processed = _process_one_car(page, plate, current_node, flow_name, processed, failures)
    
    return processed, True


def _scan_workbench(page):
    """扫描工作台，返回 (plate, current_node, flow_name) 列表"""
    goto_workbench(page)
    main_kef = get_main_kef(page)
    # ... 扫描逻辑
    return workbench_plates
```

**测试方式：**
1. 准备一个工作台有未完成任务的场景（用 DEBUG=True + 中断恢复）
2. 跑 `python run.py`，观察是否能正常续跑

**回退：** 还原 main() 里的旧代码块，删掉新函数

### Step 3.2：抽 `_process_one_car()`

**当前 main() 里 "处理一辆车" 的逻辑：**
```python
# 失败计数 + dispatch 调用 + 错误处理
failures[plate] = failures.get(plate, 0) + 1
...
try:
    dispatch(page, plate, run_from_step, processed)
    print(f"\n  ✅ {plate} 全部流程完成")
    failures.pop(plate, None)
except Exception as e:
    failures[plate] = failures.get(plate, 0) + 1
    ...
    if fc >= config.MAX_FAIL:
        SKIP_PLATES.add(plate)
        print(f"  ⏭ {plate} 已加入黑名单,不再自动重试")
```

**重构为：**
```python
def _process_one_car(page, plate, current_node, flow_name, processed, failures):
    """处理一辆车（含重试 + 黑名单逻辑）"""
    step_map = {
        config.NODE_VEHICLE_CHECK: config.STEP_VEHICLE_CHECK,
        ...
    }
    run_from_step = step_map.get(current_node, "unknown")
    processed += 1
    print(f"\n{'='*55}")
    print(f"  第 {processed} 辆: {plate}")
    try:
        dispatch(page, plate, run_from_step, processed)
        print(f"\n  ✅ {plate} 全部流程完成")
        failures.pop(plate, None)
    except Exception as e:
        failures[plate] = failures.get(plate, 0) + 1
        fc = failures[plate]
        print(f"\n  ❌ {plate} 处理失败 ({fc}/{config.MAX_FAIL}): {e}")
        screenshot_on_error(page, plate)
        _close_all_popups(page)
        if fc >= config.MAX_FAIL:
            SKIP_PLATES.add(plate)
            print(f"  ⏭ {plate} 已加入黑名单,不再自动重试")
    pa(1.5)
    return processed
```

**回退：** 把这段逻辑复制回原位置

### Step 3.3：抽 `_process_new_plate()` 

**处理新车的逻辑：**
```python
# 导航到普货审验 + 取车牌 + dispatch
plate = get_next_plate_from_list(page)
if not plate:
    # 重试逻辑
    ...
processed = _process_one_car(page, plate, "", "", processed, failures)
```

**重构为：**
```python
def _process_new_plate(page, processed, failures):
    """去列表取新车并处理
    Returns: (processed, should_break)
    """
    # 导航
    contents = get_contents(page)
    try:
        _click_expand_freight_manage(contents)
        _click_normal_review_link(contents)
        if not _verify_in_normal_review(page):
            _click_normal_review_link(contents)
    except Exception as e:
        print(f"  普货审验自动导航失败: {e}")
        input("  >>> 请手动点左侧[普货审验],然后按回车...")
    
    # 取车牌
    plate = get_next_plate_from_list(page)
    if not plate:
        return processed, False  # 列表空
    
    # 黑名单检查
    if plate in SKIP_PLATES:
        print(f"  ⏭ 跳过 {plate} (已被拉黑)")
        return processed, False
    
    # 处理
    processed = _process_one_car(page, plate, "", "", processed, failures)
    
    # 单次/上限检查
    if config.SINGLE_RUN:
        print("\n  🛑 单次模式,已处理 1 辆,停止")
        return processed, True  # 停止
    if config.MAX_CARS > 0 and processed >= config.MAX_CARS:
        print(f"\n  🛑 已达上限 {config.MAX_CARS} 辆,停止")
        return processed, True  # 停止
    
    return processed, False
```

**回退：** 还原 main() 里的旧代码
